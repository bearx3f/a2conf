# a2conf

## cmd package

### `a2fmt`

format apache configuration file with options

```
Usage of %GOPATH%\bin\a2fmt.exe:
  -file string
        Formatting file
  -tab2space
        Convert to tab to space for indent.
  -tabsize int
        indent space size (default 4)
```

#### Installation

```
go get -u beargit.pw/booboo/a2conf/cmd/a2fmt
```
